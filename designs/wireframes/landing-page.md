# 🌐 Resonance Landing Page — Wireframe Layout (English Version)

## 1. Overview

Design a landing page for a platform named:

> **Resonance**
> Tagline: **"Let lives influence lives"**

### Core Concept

A global platform for sharing life stories and human experiences through **story cards**, enabling emotional resonance between people.

---

## 2. Layout Structure (Top → Bottom)

```
[ Header ]
[ Hero / Intro ]
[ Story Card Feed ]
[ Optional CTA Section ]
[ Footer ]
```

---

## 3. Header

### Layout

```
[ Logo ]        [ Navigation ]        [ Account ]
```

---

### Components

#### (a) Left — Logo

* Clickable (redirects to homepage)
* Minimal style
* Optional soft organic accent behind logo

---

#### (b) Center — Navigation

* About
* Explore
* Stories

Style:

* Minimal
* Clean spacing
* No heavy decoration

---

#### (c) Right — Account Area

* Avatar or profile icon
* Rounded shape
* Optional subtle organic background

---

### Style Notes

* Background: soft warm tone (cream / light beige)
* Optional: subtle wave divider at bottom
* Keep header visually light

## 8. Illustration & Placeholder Rules

### When no image assets are available:

👉 Always use placeholders instead of leaving empty space

---

### Placeholder Types

#### 1. Organic Blob Placeholder

* Irregular shape
* Soft pastel fill
* Subtle grain texture

---

#### 2. Frame Placeholder

* Rounded rectangle
* Light border
* Soft noise texture

---

#### 3. Reserved Illustration Space

* Empty but structured
* Maintains layout balance

---

### Rule

> Placeholders must follow the same organic visual language
> (never use plain gray boxes)

---

## 9. Design Constraints

* Max 2–3 organic shapes per section
* Do not distort core UI components
* Maintain strong readability
* Prioritize usability over decoration
