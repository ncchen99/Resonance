# 05 — My Card Box (自己的個人空間)

**對應:** M7 思想地圖 (簡化版) + M8 私人卡片 · **路由:** `/[locale]/me`

**核心哲學:** 這是你的第二大腦,不是你的「作品集」。沒有數字炫耀,只有思維的沉澱軌跡。

## Profile Header

- Avatar 72px (`HandDrawnAvatar` 放大版)
- 筆名 + 驗證勾 (`HandDrawnCheckmark`,綠色 sage tint)
- 語言偏好:幫助閱讀者知道原文傾向
- 加入時間(非社交指標,只是時間感)
- Bio: 一句話,max 80 字

**刻意不顯示:**
- 粉絲/連結人數
- 總卡片數的大數字
- 「本週熱門」之類排行

## 五個 Tab

### Tab A — 已發布 (預設)
公開卡 grid,StoryCard 標準樣式,6 色循環。
角落顯示發布範圍圖示:🌐 公開 / 🔗 僅連結。

### Tab B — 私人 (M8)
只有自己看得到,StoryCard 改用 灰度色調 (ghost variant)。
上方 banner:
> 🔒 這些卡片只有你看得到。隨時可以改為公開。

### Tab C — 草稿
未發布的 WIP,縮小卡片形式,顯示 last edited。
[ 繼續寫 ] / [ 丟棄 ] 動作。

### Tab D — 共振過的 (取代 bookmarks)
我按過 🌊 的其他人的卡。
以時間軸排列,附「當時你為什麼共振?」可選 1 行筆記(只有自己看)。

### Tab E — 思想地圖 (Phase 2)

```
┌────────────────────────────────────────────────────────────────┐
│                                                                │
│           ○ 成長                                               │
│        ○──○──○                                                 │
│       /       \                                                │
│      ○         ○ 孤獨                                          │
│      │         /                                               │
│      ○───────○                                                 │
│     創作     關係                                              │
│                                                                │
│  [ 縮放 + / - ]   [ 聚類重算 ]   [ 匯出 ]                      │
└────────────────────────────────────────────────────────────────┘
```
- 有機手繪風格的 force-directed graph
- 節點 = 卡片,邊 = AI 判定的概念相近度
- 點節點預覽 → 點節點 title 進詳情
- AI 自動標出「核心思維聚落」(聚落命名由 AI 建議,使用者可改)
- 付費功能入口:「讓 AI 幫我把『成長』這 9 張串成一篇文章」
