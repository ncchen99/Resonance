# 03 — Card Detail

**對應:** M5 共振標記 · **路由:** `/[locale]/card/[id]`

**核心哲學:** 詳情頁不是「閱讀 + 留言」,是「閱讀 + 選擇如何回應」。三條回應路徑:共振、寫一張卡回應、發起連結。無公開留言區。

## 元件規格

**作者列 (authorRow)**
- `HandDrawnAvatar` 40px, 初始字母, 使用卡片 accent hue
- 筆名 + 地區旗/語言代碼 + 驗證勾(綠色手繪 check)
- **不顯示連結數、卡片總數、粉絲數**

**卡面外框**
- 單張詳情時 border 使用 `HandDrawnBorder` (chalk 質感, seed = card.id.hashCode)
- fill `var(--color-card-bg)`, 內距 clamp(32px, 5vw, 56px)
- 最大寬度 720px,置中

**思維核心 pull-quote**
- `Playfair Display 28px 700`, terracotta accent
- 兩側有手繪 ❝ ❞ 引號 SVG
- 下方 wavy divider 分隔故事本體

**翻譯列**
- 使用者母語若不是原文,預設看翻譯版;點「看原文」切換
- 翻譯標示:`這是 AI 翻譯 · 原文為 繁體中文 · 譯文可能損失細微語氣`

## 回應路徑 (取代按讚與留言)

### A. 🌊 共振 (M5)
- 主按鈕,發光的 terracotta pill
- 點擊後按鈕變為「已共振」,顯示:
  > 作者會知道有人共振了,但看不到是誰(除非雙方已連結)
- 使用者自己可在「我共振過的卡」收藏清單裡找回

### B. 寫一張卡回應 (引用回應)
- 跳至 `card-create` 頁面,預填 "回應 @小柚 的《勇敢不是…》"
- 新卡會自動帶 reference link,作者可在自己的私人通知看到

### C. 發起連結 (M6)
- 若雙方尚未連結 → 開啟 [07 connection-invite 模態](07-connection-invite.md)
- 若已連結 → 按鈕變為「私訊 @小柚」

### 作者自己看自己的卡
額外顯示一個只有自己看得到的摘要面板:

```
┌────────────────────────────────────────────────┐
│  這張卡今天:                                  │
│  ·  被 34 個人讀完                              │

│  ·  有 2 個人想和你連結                         │
│                                                │
│  (只有你看得到這些數字,系統不公開。)          │

```

## 空與邊界狀態

| 狀態 | 呈現 |
|---|---|
| 卡片被作者改為私人 | 顯示:「這張卡片已被作者收回」(不顯示原內容) |
| 作者已連結 | 「發起連結」 → 「私訊 @筆名」 |
| 自己是作者 | 隱藏共振與連結按鈕,顯示「編輯 / 調整發布範圍 / 刪除」 |
| 延伸卡未有結果 | 不顯示延伸區塊,直接接下一張日常卡片推薦 |
