# 02 — Home / Resonance Feed

**對應:** M4 同頻匹配 · **路由:** `/[locale]/home`

**核心哲學:** 這不是無限滑動的 feed,而是「今天 AI 為你挑選的 N 張共振卡片」。每日一批,刷新感有節制,鼓勵深讀而非滑動。

## 關鍵設計決策

**1. 每日配額感 (非無限 feed)**
- 底部不是「載入更多」,是「今天先到這裡」
- 「再給我 3 張」是溫和放寬,但仍有上限(每日最多 30 張)
- 目標:對抗多巴胺 loop,回到「卡片盒式沉澱」

**2. 匹配透明性**
- 每張卡頂端的 TAG 是「AI 認為你和這張的共鳴點」
- Hover 卡片 → 顯示 tooltip:「因為你寫過 3 張關於『成長』的卡片」
- 點擊 tag → 進入該 tag 的同頻分群(不是趨勢榜)

**3. 沒有數字社交指標**
- 不顯示「共振數」、「閱讀數」
- 作者只顯示筆名 + 地區/語言(強化「世界有人懂我」)
- 不顯示任何排序序號

## 互動狀態

| 元素 | 狀態 | 行為 |
|---|---|---|
| StoryCard | hover | 背景從 `oklch(97.5% 0.012 {hue})` → `oklch(92.5% 0.024 {hue})`, arrow icon opacity 0.28 → 0.7, radial mask 460ms |
| StoryCard | click | push to `/card/{id}` |
| 🔔 鈴鐺 badge | 未讀 > 0 | 顯示紅色小圓點 (terracotta),無數字 |
| 「再給我 3 張」 | click | 追加注入 3 張卡,當日配額扣掉 |

## Mobile (≤720px)

- StoryCard 進入 full-bleed 模式 (見 DESIGN.md §5.3 Mobile)
- 卡片間以 wavy divider (amplitude 1.4, steps 7) 分隔
- 全域 nav 收進 hamburger,右下角 floating `OrganicButton primary 寫卡片 +`
