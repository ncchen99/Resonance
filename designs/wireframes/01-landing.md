# 01 — Landing (訪客首頁)

**狀態:** 已實作於 [app/[locale]/page.tsx](../../src/app/[locale]/page.tsx)

**對象:** 未登入訪客。目標是讓人在 5 秒內理解「這不是另一個社群平台」。

## 結構

```
SiteHeader
   │
HeroSection
   │   (SectionEdge wavy)
CardFeedSection
   │   (SectionEdge wavy)
CTASection
   │   (SectionEdge wavy)
SiteFooter
```

## 1. SiteHeader

```
┌────────────────────────────────────────────────────────────────────────┐
│                                                                        │
│  Resonance          共振是什麼    卡片示例    關於   [登入]  [開始]    │
│                                                                        │
├─────── ∿∿∿ wavy line ∿∿∿ ─────────────────────────────────────────────┤
```

- 訪客態:右側 `[登入] ghost` + `[免費開始] primary`
- 捲動後 header bg 從透明 → `oklch(96% 0.015 75 / 0.92)` + blur

## 互動筆記

- Hero 按鈕皆有 hover radial reveal (340ms)
- 訪客點任一卡片 → routing 至 `/[locale]/signin?next=/card/{id}`
- 語言切換器保留在 footer;header 不放,避免干擾
