# 04 — Card Create / Edit

**對應:** M1 卡片建立 + M2 AI 寫作輔助 · **路由:** `/[locale]/write` (新增) / `/[locale]/write/[id]` (編輯)

**核心哲學:** 結構化的極簡模板,把人拉離「長文思維」。AI 是夥伴,不是代筆人。

## 模板欄位 (與 M1 一致)

| 欄位 | 類型 | 限制 | 必填 |
|---|---|---|---|
| 思維核心 | 單行輸入 | 60 字 | 是 |
| 故事本體 | 多行編輯器 (plain text, 無富文本) | 300–800 建議,hard cap 1200 | 是 |
| 標籤 | TagPill 陣列 | 最多 6 個,每個 2–8 字 | 否 (AI 會幫生) |
| 媒體 | 圖片 1 張或短影片 ≤15s | ≤ 8MB | 否 |
| 發布範圍 | 單選 | 私人 / 僅連結 / 公開 | 是 (預設公開) |
| 原文語言 | 單選 | 五語 | 是 (自動偵測 + 可改) |
| 自動翻譯 | 勾選 | 預設勾 | 否 |

## M2 AI 寫作輔助 — 三個動作

**1. 幫我潤稿**
- 觸發方式:按鈕 / 寫完停頓 5 秒後 AI 提示「要不要幫你順一下?」
- 呈現方式:diff mode,逐句比對,使用者逐段 [採用] / [保留原樣]
- 不破壞語氣:prompt 要包含「保持作者個人語氣、節奏、用字偏好」

**2. 幫我下標 (思維核心建議)**
- 讀完故事後給 3 個候選句子
- 使用者 [採用] / [微調] / [重新生成]

**3. 生成標籤**
- 根據思維+故事產出 3–5 個標籤
- 使用者可 [✕ 刪除] / [+ 加自訂]
- 標籤是匹配關鍵,所以 AI 的產出要偏向「思維概念」而非關鍵字
  - ✓ `TAG·脆弱性` `TAG·世代創傷`
  - ✗ `TAG·面試` `TAG·阿嬤`

## 字數警示 (溫和,不阻擋)

| 字數 | 狀態 | 呈現 |
|---|---|---|
| < 300 | 提醒 | 「短一點沒關係,但故事能讓共振更強」 (muted text,不阻擋) |
| 300–800 | 理想 | 字數計變成暖色勾勾 |
| 800–1200 | 警示 | 「共振卡片不鼓勵長文。要不要試試拆成兩張?」 + [拆成兩張 AI 建議] |
| > 1200 | 硬上限 | 無法再輸入,提示拆卡 |

## 自動儲存 & 草稿

- 每 10 秒 autosave 到 draft
- 離開頁面有未存內容 → modal 確認
- 我的卡片盒 > 草稿 tab 可看到所有未發佈卡

## 發布後

- toast:「已發布。AI 正在為你翻譯中(約 30 秒完成)」
- 翻譯完成 → 通知 🔔 鈴鐺
- 導向 [03 card-detail](03-card-detail.md) 的作者自己視角

## Mobile (≤720px)

- AI 面板改為底部 sticky bar,點「✨」展開 action sheet
- 編輯器 full-bleed,無卡片外框(避免擠壓輸入空間)
- 發布範圍 + 語言折為「進階設定 ▾」
